She Can Foundation Contact Form

A responsive and user-friendly contact form created using HTML, CSS, and JavaScript as part of the She Can Foundation Internship Task.

Features

- Name, Email, and Message fields
- Form Validation
- Success Message on Submission
- Local Storage Support
- Responsive Design
- Modern UI with Animations
- Mobile Friendly

Technologies Used

- HTML5
- CSS3
- JavaScript

Project Preview

The contact form allows users to enter their details and submit messages with proper validation. A success message is displayed after successful submission.

How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open "index.html" using Live Server or any browser.
4. Fill out the form and click Submit.
